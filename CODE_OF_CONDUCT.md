# Code of Conduct

## Our Standards

We are committed to providing a professional, respectful, and productive environment for everyone contributing to Scutum's open-source projects.

### Expected Behavior

- Professional and respectful communication
- Constructive feedback focused on code and architecture
- Respect for differing viewpoints and approaches
- Focus on what is best for the project and its users
- Accountability for mistakes and willingness to learn

### Unacceptable Behavior

- Harassment, intimidation, or discrimination of any kind
- Personal attacks or inflammatory language
- Publishing others' private information without consent
- Any conduct that would be considered inappropriate in a professional setting

## Scope

This code of conduct applies to all Scutum open-source repositories, issue trackers, pull requests, and community interactions.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers. All complaints will be reviewed and investigated promptly and fairly.

Project maintainers have the right to remove, edit, or reject contributions that violate this code of conduct.
