# .github

Organization-wide engineering standards for [Scutum Defense](https://github.com/scutum-defense).

## What this repo provides

All Scutum repositories inherit these defaults unless they define their own:

- **Pull request template** — standardized review format
- **Issue templates** — feature request, bug report, security review, ADR request
- **Security policy** — vulnerability reporting and responsible disclosure
- **CODEOWNERS** — review gate enforcement
- **Org profile** — public overview at github.com/scutum-defense
