# ScutumDefense Org Standards

This repository contains organization-wide standards, templates, and baseline workflows for Scutum.

## Purpose

Use this repo to define:
- pull request standards
- issue templates
- security reporting guidance
- default ownership rules
- baseline workflow hygiene

## Rules

- all critical repos are private
- direct pushes to protected branches are not allowed
- CODEOWNERS review is mandatory where configured
- no secrets or credentials may be committed
- every repo must have a README and CODEOWNERS file
